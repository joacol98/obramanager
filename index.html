Cómo configurar Firebase (gratis)
1. Entrá a console.firebase.google.com
2. Crear proyecto → poné un nombre → sin Analytics
3. Build → Realtime Database → Create database → Start in test mode
4. Project Settings (engranaje) → General → copiá Project ID y Web API Key
5. Pegá esos datos arriba y listo
